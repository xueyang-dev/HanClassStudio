# Courseware Runtime

The current renderer emits a self-contained `lesson.html` with inline Fresh Classroom styles and interaction runtime. This folder documents the intended source home for shared courseware themes, static runtimes, and template assets as the renderer grows.

Implemented v0.1 interactions:

- AudioButton
- VocabularyFlipCard
- SentenceDragBuilder
- ListenAndChoose
- MatchGame
