# Screenshot Checklist

Use these screenshots for a portfolio case study or project page.

| Screenshot | What To Show | Value | Suggested Caption |
|---|---|---|---|
| Workbench landing/upload | Sidebar workflow and upload area | Shows the teacher-facing entry point | "Start from familiar PPTX/PDF materials." |
| Course profile confirmation | Lesson title, learner level, scaffolding language, generation mode | Shows teacher control before generation | "Teacher confirmation gates the pipeline before AI generation." |
| Pipeline running or completed | Spec Lock, Blueprint, Media, Render, Quality, Export statuses | Shows this is a staged pipeline | "Artifact-first generation with explicit pipeline phases." |
| Preview runtime | Slide-based `lesson.html` with large classroom UI | Shows output quality | "Projection-ready interactive HTML courseware." |
| Language mode toggle | Chinese / scaffold / bilingual mode | Shows language pedagogy | "Chinese remains core; scaffolding supports comprehension." |
| Interactive component | Vocabulary flip card or sentence builder | Shows interactivity beyond static slides | "Runtime components support classroom practice." |
| Quality panel | Quality state, warnings, passed checks | Shows gatekeeping | "Quality gate prevents unsafe normal export." |
| Spec Lock Summary | Route, generation mode, runtime template, allowed components | Shows execution contract | "Spec lock makes generation policy inspectable." |
| Artifact Inspector | `specs/`, `blueprints/`, `courseware/`, `quality/`, `exports/`, `agent/` | Shows artifact ownership | "Every stage writes canonical artifacts." |
| Agent Handoff panel | Generated Agent task and validate button | Shows agent-compatible workflow | "External Agents edit artifacts; HanClassStudio validates and renders." |
| Export ZIP contents | `lesson.html`, `assets/data`, quality report, manifest | Shows deliverable | "Offline export includes runtime, assets, data, and quality report." |

## Capture Notes

- Use a clean project with a readable HSK 1 or beginner Chinese lesson title.
- Prefer browser width around 1440 px for workbench screenshots.
- Capture runtime preview in fullscreen or near-fullscreen.
- Keep file paths visible when showing Artifact Inspector.
- Avoid screenshots with loading spinners unless the point is pipeline progress.
